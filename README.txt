Version: 0.19 alfa

This application reads jpg files with photos of barcodes and
retrieves the book data from the Web. Read pinaker.pdf for
more details.

IMPORTANT: Before running pinaker on Linux you need to 
create a home/.config/pinaker folder and copy there the 
*.pws files in the pinaker/websources folder and the 
htmlcodes.txt file in the pinaker folder.
